# SIMPLE BLOG APP

```
    . Is a test app, where real user are able to post their blogs
    . Implementing docker, aws, jenkins
```

## Technology

```
Node.js, expressJs, mongoDb, pug, bootstrap
Docker container,
AWS ECR
REST API
JENKINS
```

## Installation

    ```
       $ git clone https://github.com/amritregmi/blog-app
       $ npm install
    ```

## Usage

```
To run in development
    npm run start:dev

```
